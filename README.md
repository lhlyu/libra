# libra
go iris12

### 版本说明

- go版本: v1.13+
- iris版本: v12.1.8

### 所需环境

- go
- git


### 傻瓜式入门

- 第零步克隆到自己电脑

`git clone https://github.com/lhlyu/libra.git`

- 第一步设置代理

`go env -w GOPROXY=https://goproxy.cn,direct`

- 第二步进入工程目录

`cd libra`

- 第三步运行

`go run main.go`

- 第四步，浏览访问

`http://localhost:8080`

- 注，goland打开爆红处理

1. 快捷键打开goland设置

`Ctrl + Alt + S`

2. 按图修改

![示例](https://github.com/lhlyu/libra/blob/master/img/1.png?raw=true)
